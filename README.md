BMI Calculator 🏋️

Overview

This project is a simple Python-based BMI Calculator that calculates a user's Body Mass Index (BMI) using their weight and height.

It also classifies the BMI into different health categories.

---

Features

- Accepts user input for weight and height
- Calculates BMI using the standard formula
- Displays BMI result with 2 decimal places
- Categorizes BMI into:
  - Underweight
  - Normal Weight
  - Overweight
  - Obese
- Validates input values
- Handles invalid input errors

---

Technologies Used

- Python

---

BMI Formula

BMI = Weight (kg) / Height² (m²)

---

How to Run

Step 1: Save the file

bmi_calculator.py

Step 2: Run the program

python bmi_calculator.py

---

Example

Input

Enter your weight in kilograms: 70
Enter your height in meters: 1.75

Output

===== BMI Result =====
Your BMI is: 22.86
Category: Normal Weight
